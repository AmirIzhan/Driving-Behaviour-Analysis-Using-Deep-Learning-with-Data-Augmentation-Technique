# Dataset — UAH-DriveSet

This project uses the UAH-DriveSet, a public driving behaviour dataset.

## Download
Official source: http://www.robesafe.uah.es/personal/eduardo.romera/uah-driveset/

## After Downloading
Place the extracted folders inside  data/raw/
Expected structure:
   data/raw/D1/trip_folder_1/RAW_ACCELEROMETERS.txt
   data/raw/D1/trip_folder_1/RAW_GPS.txt
   data/raw/D2/...

## Classes
0 = Aggressive   1 = Drowsy   2 = Normal
